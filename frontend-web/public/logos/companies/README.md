## Formato recomendado

- Tipo: PNG con fondo transparente (preferido) o SVG.
- Tamano base sugerido: 256x256 px o 512x512 px.
- Peso recomendado: <= 120 KB por archivo.
- Espacio de color: sRGB.

## Uso en app

- Backend publica `logo_path` en `/api/prioritized-clients`.
- Frontend usa `logo_path` en selector y cabecera.
