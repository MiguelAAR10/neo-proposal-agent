# RESUMEN PARA USUARIO - COMO USAR EL MVP V2

## Que es esto

NEO Proposal Agent V2 te ayuda a construir una propuesta comercial en minutos con casos reales, no desde cero.

## Flujo rapido de uso

1. Ingresa empresa, rubro, area y problema.
2. Elige tipo de casos: NEO, AI o Ambos.
3. Revisa y selecciona casos sugeridos.
4. Genera propuesta inicial.
5. Ajusta texto (refinamiento) y exporta.

## Que debes esperar

- Respuesta rapida de casos relevantes.
- Propuesta estructurada y clara.
- Referencias a casos usados.
- Menor tiempo de preparacion.

## Buenas practicas para mejores resultados

- Describe el problema con detalle concreto (objetivo + dolor + impacto).
- Selecciona 2-4 casos realmente alineados.
- Usa refinamiento para ajustar tono y foco ejecutivo.

## Limites del MVP

- Algunas funciones admin/avanzadas pueden estar fuera del MVP inicial.
- La calidad depende de la calidad del problema ingresado y de los casos seleccionados.

## Errores comunes y como actuar

- "No hay casos disponibles": solicitar ingesta de datos al admin.
- "No encontramos casos similares": reformular problema con mas contexto.
- "Error de generacion": reintentar; si persiste, revisar estado de servicios.

## Checklist antes de cerrar una propuesta

- [ ] Problema del cliente claramente definido.
- [ ] Casos seleccionados con evidencia suficiente.
- [ ] Propuesta final en tono adecuado para el decisor.
- [ ] Exportacion validada (texto/PDF).
